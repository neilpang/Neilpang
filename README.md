## About Me

I'm a systems-oriented developer with deep expertise in low-level and infrastructure code. Since joining GitHub in 2014, I've focused on building practical, production-ready tools — primarily in **C**, **C++**, and **Shell**, with additional work across the full systems stack.

My public work centers on automation, networking, security, and simplifying complex deployments. I contribute heavily as a **builder**, with over 95% of my activity in direct commits. Currently active in **Shell**, **Go Template**, **PowerShell**, **JavaScript**, **Dockerfile**, **C**, **CMake**, and **Java**.

I'm the author of:

- **[acme.sh](https://github.com/acmesh-official/acme.sh)** — the popular pure-shell ACME client
- **[VMActions.org](https://vmactions.org)**
- **[AnyVM.org](https://anyvm.org)** — enabling "run any VM anywhere"
- **[Babelive.org](https://babelive.org)**

**Recent activity (last 90 days):** 52 active days across 4 public repositories.
